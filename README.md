# Bibi & Fufu Codex Pet

以 Bibi 和 Fufu 为原型制作的 Codex v2 动态宠物。宠物仅保留两位角色，包含待机、左右移动、挥手、跳跃、失败、等待、运行、检查和 16 个视线方向动画。

![Bibi & Fufu 动画预览](preview.png)

## 安装

1. 下载本仓库。
2. 将整个仓库目录复制到：

   ```text
   %USERPROFILE%\.codex\pets\bibi-and-fufu
   ```

3. 确认目录内至少包含 `pet.json` 和 `spritesheet.webp`。
4. 重新打开 Codex，在宠物选择中启用 **Bibi and Fufu**。

## 文件

- `pet.json`：Codex v2 宠物配置。
- `spritesheet.webp`：1536 × 2288、8 × 11 的透明动画图集。
- `preview.png`：全部动画状态预览。
